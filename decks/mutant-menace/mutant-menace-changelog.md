# Changelog — The Wise Mothman (Mutant Menace)

This file is the append-only session ledger for this deck. Each session gets one entry block.
New entries are added at the top. Do not edit or delete prior entries.

## How to Use

- **Proposed:** Swaps identified during a session but not yet executed in Moxfield
- **Confirmed:** Swaps executed in Moxfield and physically made (or queued for physical update)
- **Rejected:** Cards considered and explicitly cut from future consideration — do not suggest again

The AI reads this file at the start of every session. Confirmed and Rejected entries are binding.
